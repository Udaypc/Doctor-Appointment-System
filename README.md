## üß© Microservices Used

  - Handles authentication and authorization

- **Auth Service**

  - Token generation and validation

- **Patient Service**
  - Patient registration
  - Appointment booking
  - Profile management

- **Doctor Service**
  - Doctor data
  - Appointment handling

- **Eureka Server**
  - Service registry and discovery

## üîê Security Flow

1. User logs in using OAuth2.
2. Token is generated by Auth Service.
3. Client sends token with each request.
4. API Gateway validates token.
5. Request is routed to appropriate microservice.
